# Руководство пользователя CharacterBook

## 🎯 Начало работы

### Создание первого персонажа

1. **Откройте приложение** и нажмите кнопку "+"
2. **Выберите шаблон** - стандартный или пользовательский
3. **Заполните основную информацию**:
   - Имя и фамилия персонажа
   - Раса
4. **Добавьте детали**:
   - Внешность и описание
   - История персонажа
   - Способности и навыки
5. **Сохраните** персонажа

### Основной интерфейс

**Главный экран:**
- 📋 Список персонажей
- 🔍 Поиск и фильтрация
- 📁 Организация по папкам
- ⚙️ Настройки приложения

**Экран персонажа:**
- ✏️ Редактирование информации
- 📷 Добавление изображения
- 📤 Экспорт в PDF
- 🗑️ Удаление персонажа

## 👥 Управление персонажами

### Создание и редактирование

**Базовые поля:**
- Основная информация
- Характеристики
- Навыки и способности

**Расширенные поля:**
- История и предыстория
- Отношения с другими персонажами
- Цели и мотивация
- Заметки и дополнительная информация

### Организация персонажей

**Папки и категории:**
- Создавайте папки для кампаний
- Группируйте по игровым системам
- Используйте теги для быстрого поиска

**Поиск и фильтрация:**
- Поиск по имени и характеристикам
- Фильтрация по различным атрибутам
- Сортировка по дате создания/изменения

## 🏷️ Расы и шаблоны

### Стандартные шаблоны (скоро)

**Включают:**
- D&D 5e
- Pathfinder
- Универсальный шаблон
- Пустой шаблон (для кастомизации)

1. **Создайте новый шаблон**
2. **Добавьте нужные поля**:

### Управление расами

**База рас:**
- Предустановленные описания рас

## ❓ Частые сценарии

**Как перенести персонажа на другое устройство?**
- Экспортируйте в PDF и импортируйте на новом устройстве
- Используйте функцию бэкапа

**Как создать сложного персонажа с кастомными полями?**
- Используйте пустой шаблон
- Добавьте нужные поля вручную
- Сохраните как новый шаблон для повторного использования

**Как организовать много персонажей?**
- Используйте папки для разных кампаний
- Применяйте теги для категоризации
- Используйте поиск и фильтрацию
