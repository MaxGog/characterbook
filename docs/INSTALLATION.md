# Установка и настройка CharacterBook

## 🎯 Быстрая установка для пользователей

### Из официальных магазинов:

**📱 Android:**
- [Google Play](https://play.google.com/store/apps/details?id=ru.maxgog.listcharacters&hl)
- [RuStore](https://www.rustore.ru/catalog/app/ru.maxgog.listcharacters)

**💻 Windows:**
- [Microsoft Store](https://apps.microsoft.com/detail/9NKV4DBQJW0S)

**🌐 Веб-версия (скоро):**
- Доступна онлайн (требуется современный браузер) (скоро)

### Системные требования:
- **Android**: 5.0+ (50 МБ свободного места)
- **Windows**: Windows 10+ (100 МБ свободного места)
- **Web**: Chrome 90+, Firefox 88+, Safari 14+

## 🛠 Установка для разработчиков

### Предварительные требования:
- Flutter SDK 3.0.0+
- Dart SDK 3.7.2+
- Git

### Шаги установки:

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/maxgog/characterbook.git
cd characterbook

# 2. Установите зависимости
flutter pub get

# 3. Запустите приложение
flutter run
```

#### Сборка для разных платформ:

```bash
# Android
flutter build apk

# Windows
flutter build windows

# Web
flutter build web
⚙️ Первоначальная настройка
```

## После установки:
1. Выберите язык - русский или английский
2. Выберите тему - светлая, тёмная или системная
3. Создайте первого персонажа - используйте мастер создания
4. Ознакомьтесь с шаблонами - выберите подходящий шаблон

### Настройки приложения:

1. Внешний вид: Тема, цветовая палитра
2. Хранение: Настройки резервного копирования
3. Экспорт: Форматы и настройки PDF

### 🔄 Обновление

#### Автоматическое:

- Магазины приложений обновляют автоматически
- Веб-версия обновляется автоматически

#### Вручную:

- flutter pub get для обновления зависимостей

## 🐛 Решение проблем

### Частые проблемы:

#### Приложение не запускается:

1. Проверьте совместимость версии ОС
2. Переустановите приложение
3. Освободите место на устройстве
4. Очистите кеш приложения

#### Проблемы с установкой для разработчиков:

```bash
# Очистите кэш
flutter clean

# Проверьте окружение
flutter doctor

# Обновите зависимости
flutter pub upgrade
```

### Проблемы с экспортом:

1. Проверьте разрешения на запись
2. Убедитесь в наличии свободного места
