# Возможности CharacterBook

## 🎯 Основные функции

### Управление персонажами
- **Создание и редактирование** - интуитивный редактор персонажей
- **Структурированные шаблоны (скоро)** - готовые формы под популярные RPG системы
- **Гибкие поля (скоро)** - текстовые, числовые, выпадающие списки
- **Изображения персонажей** - добавление визуального представления

### Организация данных
- **Папки и категории** - систематизация по кампаниям и играм
- **Поиск и фильтрация** - быстрый поиск по всем полям
- **Сортировка** - по имени, дате создания, уровню
- **Теги и метки** - дополнительная категоризация

### Шаблоны и настройки
- **Предустановленные шаблоны (скоро)** - D&D, Pathfinder, универсальный
- **Конструктор шаблонов** - создание собственных форм
- **База рас (скоро)** - стандартные расы с описаниями
- **Пользовательские расы** - расширение базы данных

## 💾 Работа с данными

### Экспорт и печать
- **PDF экспорт** - генерация листов персонажей
- **Настраиваемые форматы** - выбор включаемых полей
- **Печать** - оптимизация для бумажных носителей
- **Совместное использование** - отправка мастеру или игрокам

### Хранение и безопасность
- **Локальное хранение** - полная оффлайн работа
- **Автосохранение** - защита от потери данных
- **Резервное копирование** - регулярные бэкапы (скоро)
- **Безопасность** - локальное шифрование данных
